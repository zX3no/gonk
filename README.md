## üì¶ Installation

> I recommend using a font with ligatures for the best experience.

#### Dependencies

Debian:

```
sudo apt install libasound2-dev pkg-config build-essential
```

Alpine:

```
apk add pkgconf alsa-lib-dev alpine-sdk
```

Arch:

```
pacman -S pkgconf alsa-lib base-devel
```

#### crates.io

```
cargo install gonk
```

Add your music:

```
gonk add D:/Music
```
